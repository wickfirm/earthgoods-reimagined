# Earth Goods — Reimagined

A modern e-commerce storefront for Earth Goods, a premium organic food brand delivering across the UAE.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

Make sure you have Node.js & npm installed — [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm i

# Start the development server
npm run dev
```

## Editing

You can edit files directly in GitHub or clone the repo and push changes from your preferred IDE. You can also use GitHub Codespaces for an in-browser development environment.

## Deployment

Deploy via Vercel, Netlify, or any static hosting provider by running `npm run build` and serving the `dist` folder.
